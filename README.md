## 功能
* 通过密码自动登录 A-3000 网站 (我们不会收集您的账号与密码)
* 当 水平测试界面 弹出时 点击 开始测试答题
* 当 进入主页 时 等待
* 当 阅读选择题 弹出 时 [答题](#答题)

## 答题
* 将题目与选项 复制
* 打开新标签页 打开AI网站
* 选择智能模式
* 将题目与选项粘贴,并且通过自然语言使AI思考后输出一个答案
* 处理AI输出的答案
* 回到题目标签页
* 选择AI回答的答案选项
* 点击”下一个“

## 注意事项
* 正确率大大提高了,目前经过多次人工测试,发现一般正确率处于 60% - 75%，仅在人工测试中，最低的一次测试正确率达到50%，最高的一次则可达100%
